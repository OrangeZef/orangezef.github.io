Joka.ca | Cloud Architecture & Consulting

This repository contains the source code for Joka.ca, a personal portfolio and professional IT consulting landing page.

The site serves a dual purpose:

Professional IT Consulting: A front-facing portfolio highlighting 14+ years of enterprise-grade system administration, datacenter procurement, advanced networking, and cybersecurity expertise.

Architecture Showcase: A gateway and status hub for a private, self-hosted infrastructure environment used for advanced system architecture testing and high-availability workload demonstrations.

🚀 Tech Stack

This project is built using a modern, lightweight, and highly interactive front-end stack, entirely contained within a single index.html file for rapid deployment and edge caching.

HTML5 / CSS3: Semantic structure with custom CSS keyframes.

Tailwind CSS: (via CDN) Utility-first framework for rapid, responsive UI styling and glassmorphism effects.

Alpine.js: Lightweight JavaScript framework used for state management, handling the preloader, virtual routing (for legal documents), and the slide-out command center.

Particles.js: Interactive, mouse-responsive particle network background.

AOS (Animate On Scroll): Scroll-triggered entrance animations with custom easing (ease-out-back).

Vanilla-tilt.js: Smooth 3D tilt effects with glare on capability cards.

Rellax.js: Smooth parallax scrolling effects on typography and background shapes.

✨ Key Features

Cinematic Initialization: A custom CSS preloader featuring spinning radar rings and a progress bar simulating a terminal boot sequence.

Interactive Depth: The Hero section utilizes Alpine.js to track mouse coordinates (mx, my) to create a real-time, multi-layered 3D typography effect.

Glassmorphism UI: Frost-glass panels with subtle yellow and green neon glow accents to fit the "terminal/hacker" aesthetic.

Virtual Routing: The Terms of Service and Privacy Policy are built into the single-page app architecture. Alpine.js manages state transitions to display these documents without requiring page reloads.

Fully Responsive: Fluid typography and flex/grid layouts ensure a seamless experience from mobile to ultrawide desktop monitors.

🛠️ Deployment

This site is designed to be hosted statically. It is currently deployed via GitHub Pages and mapped to a custom domain (joka.ca) via GoDaddy DNS A-records.
